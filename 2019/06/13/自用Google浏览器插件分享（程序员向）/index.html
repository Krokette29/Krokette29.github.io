自用Google浏览器插件分享（程序员向）

发表于 2019-06-13 | 分类于 自我提升 | 评论数： | 阅读次数：

谷歌浏览器插件推荐！适合程序员！

近期搜罗了一些关于提高自我学习效率的谷歌浏览器插件，整理下来之后把最终留下来用的插件分享分享。

简悦

强烈推荐！！特别适合网页阅读！尤其是像CSDN这样的，侧边的广告尤其闹心。。。可以开启阅读模式或者聚焦模式。其中聚焦模式可以把除了网页正文外的其他部分加个蒙版，避免分心。

空格之神

又名，《为什么你们就是不能加个空格呢？》。。。第一眼看上去十分的瓜皮，然而用了之后就会发现，阅读舒适度提升的不止一点半点！事实证明对于中英加数字之间加个空格可以有效提高阅读舒适度。。。

OneNote Web Clipper

遇到有用的网页剪下来吧！很早以前就有的功能，网页剪辑，然而最近才发现很好用。。。相比于印象笔记，个人还是更倾向于OneNote，没有为啥。。。就是因为OneNote用习惯了哈哈，所有笔记都在OneNote上。

阅读列表

顾名思义，把有意思的网页加入阅读列表，之后再进行进一步整理。

Custom Cursor for Chrome

看久了论文和学术文章，还是需要让生活充满乐趣的哈哈！给自己的光标换个外貌吧！

（非插件app推荐：f.lux）

f.lux不是一款谷歌浏览器插件，不过鉴于最近搜罗的过程中发现十分好用，于是也放在这里了。这是一款十分小巧的，可以自动调节屏幕色温的软件。开启之后可以根据当地的日出和日落来调节屏幕色温，让夜晚的工作更舒服，不刺眼。个人使用Classic f.lux模式，默认的Recommended Colors模式太偏黄了，不是很舒服。。。
