Hexo配置

主题更换

由于自带主题实在太丑。。所以想更换主题的话参见hexo官方主题页
下载到满意的主题之后，放入themes/文件夹下（默认有一个landscape主题），然后在主题配置文件中修改theme条目为刚刚下载的主题名称。
这里使用的是NexT主题，个人感觉很棒，之后可能会新开一篇介绍NexT主题的配置。另附上知乎的Hexo主题推荐

发布文章

使用命令hexo n [layout] 博客名字来发布文章。该命令有多种布局，默认是post，也就是直接发布新文章，文章会存储在source/_posts文件夹下。可以使用hexo n draft 博客名字来发布到草稿中，也就是存储在source/_drafts文件夹下。等到写完之后使用命令hexo publish 博客名字来发布。
另外，可以通过编辑主题配置文件中的条目new_post_name来修改Hexo的默认创建文件名称。具体参见官方文档。

文章撰写过程中的预览

Hexo的文章是用Markdown语法写的，因此直接编辑不是很直观，需要有个预览（当然熟了之后ST直接上也是可以的哈哈）。
一种方法，通过带有预览功能的md编辑器来编写。
另一种方法，命令行输入hexo s --debug来启动实时本地预览。强烈推荐！！！尤其是在鼓捣各种配置的时候超级好用！

至此，基本上Hexo能够正常进行运作了。关于Hexo进一步的配置和NexT主题的设置等我继续学习挖掘之后补上。。 。
